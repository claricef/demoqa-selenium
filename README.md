## Como executar

1. Clone o projeto
   
2. Com python e vsCode instalado na máquina, instale o ambiente virtual na pasta do projeto:
  
    python -m venv venv

3. Ative o ambiente virtual
    - PowerShell:
      
      venv\Scripts\Activate.ps1
  
    - Bash:
  
        source venv/Scripts/activate
  
  4. Instale as dependências

      pip install -r requirements.txt

  5. Execute os testes

     pytest
